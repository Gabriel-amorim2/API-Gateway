# API Gateway

Este projeto consiste na implementação de um API Gateway utilizando Spring Cloud Gateway e Spring Security WebFlux, com o objetivo de estudar conceitos de arquitetura de microserviços, segurança com JWT e controle de acesso baseado em roles.

O Gateway atua como ponto único de entrada para múltiplos microserviços de uma plataforma bancária fictícia, centralizando roteamento, autenticação e autorização.

 Objetivo 

Projeto desenvolvido exclusivamente para fins de estudo, com foco em:

Compreender o papel de um API Gateway em arquiteturas distribuídas
Implementar autenticação stateless com JWT

Aplicar autorização baseada em roles
Utilizar Spring WebFlux e programação reativa
Simular um ecossistema de microserviços bancários
